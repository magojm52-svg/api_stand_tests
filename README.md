# Pruebas para el parámetro firstName al crear un usuario/a en Urban Grocers

## Descripción
Este proyecto contiene pruebas automatizadas para validar el parámetro `firstName` al crear usuarios en la API de Urban Grocers.

## Estructura del proyecto
- `create_user_test.py` - Pruebas para la creación de usuarios
- `sender_stand_request.py` - Funciones para enviar solicitudes HTTP
- `data.py` - Datos de prueba
- `configuration.py` - Configuración de la API

## Requisitos
- Python 3.x
- pytest
- requests

```bash
pytest create_user_test.py
